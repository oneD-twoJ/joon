세부 테스트 결과 및 계획

각 기능별 테스트 상세 내용, 발견된 문제점 및 향후 조치 계획입니다. 각 항목을 클릭하여 자세한 내용을 확인하세요.

기능 설명

테스트 환경의 점진적 변화로 인한 결과 편차를 동적으로 보상하여 시스템 교정 상태를 정밀하게 유지합니다. 누설 없는 마스터 부품을 기준으로 허용 변동 범위를 설정하고, 양품들의 이동 평균을 계산하여 모든 테스트 결과를 보정합니다.

테스트 결과: 실패

  • 문제점: 테스트 중 측정값이 예상 범위를 지속적으로 벗어나는 현상 발생.
  • 추정 원인: 설정된 'EDC Percentage' 또는 'EDC Quantity' 값이 현재 테스트 환경 및 부품과 부합하지 않을 가능성.

향후 조치 계획

  • 관련 파라미터 재검토 및 조정 후 재테스트 진행
  • 제조사 매뉴얼을 참고하여 기능 초기화 조건 재확인

기능 설명

다양한 테스트에서 측정된 압력 값을 보정하여 결과의 정확도를 향상시킵니다. 테스트 중 발생 가능한 미세 압력 변동 요인을 보정하지만, 특정 환경에서는 비활성화가 필요할 수 있습니다.

테스트 결과: 실패

  • 문제점: 기능 활성화 시 오히려 측정값의 편차가 증가하는 경향 확인.
  • 추정 원인: 현재 테스트 프로그램의 특성과 보정 기능 간의 충돌 가능성.

향후 조치 계획

  • 기능 비활성화 상태 데이터와 비교 분석하여 문제 원인 파악
  • 특정 테스트 유형에 대한 기능 적용 권장 사항을 매뉴얼에서 재확인

기능 설명

여러 번의 연속적인 교정을 수행하고 결과의 평균값을 최종 교정 값으로 사용하여 정확도와 신뢰성을 높입니다. 단일 교정 시 발생할 수 있는 우발적 오차를 최소화합니다.

설정 방법

  • Batch Layout: 교정 방식 설정 (교대 또는 그룹)
  • Batch Quantity: 총 교정 횟수 설정 (2 ~ 25회)
  • Average Quantity: 평균 계산에 사용할 데이터 수 설정

사전 준비사항 (2025-07-04)

  • 테스트용 마스터 부품 및 리크 스탠다드 점검
  • 'Batch Layout' 설정에 따른 테스트 절차 사전 숙지
